## 为什么 LaunchBar 值得这个价

LaunchBar 的核心是一套自适应搜索算法。你不用记全名，输入"qcc"它能记住你想开的是 QQ 音乐，输入"tk"它知道你要 Telegram。用得越久，它越懂你。

它的功能可以拆成几块来看：

- **即时发送（Send-to）**：选中一个文件，按快捷键丢进 LaunchBar，再选目标动作——发邮件、压缩、移到指定文件夹、上传，一步到位。这是它和 Spotlight 拉开差距的关键。
- **剪贴板历史**：内置剪贴板记录器，复制过的内容随时回找，比单独装个 Paste 还省事。
- **动作执行**：搜索结果上能二次操作，比如对一张图直接"用 Preview 打开 / 压缩 / 设为桌面"。
- **脚本扩展**：支持 AppleScript、Python、Shell 脚本，开发者可以把常用工作流封装成一条指令。
- **索引自定义**：扫描范围可深度配置，既保证结果全面，又不至于把搜索栏塞满噪音。

Macworld 在其评测里给 LaunchBar 打过 Mac Gems 标签，评价是"用思维的速度控制 Mac 应用"。MacPowerUsers 社区里关于 LaunchBar vs Alfred 的讨论也很多，结论基本是：Alfred 上手快、生态热闹，LaunchBar 学习曲线略陡但一旦熟了效率天花板更高——尤其适合愿意花一周时间换一年效率的人。

## 适合谁买，不适合谁买

**适合**：

- 每天在 Mac 上处理大量文件、频繁切换应用的办公族
- 写代码、跑脚本的开发者，想把工作流压进几个字母里
- 受不了鼠标点来点去的键盘党

- 只用 Mac 看视频刷网页的轻度用户——Spotlight 够了

- Windows 用户——它只有 macOS 版，没有 Windows 版
